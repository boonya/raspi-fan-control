# Cooler

## RaspberryPI Cooling Fan Control Service

### Installation

```sh
wget https://github.com/boonya/raspi-fan-control/releases/download/0.1.0/cooler_0.1.0_arm64.deb
sudo dpkg -i cooler_0.1.0_arm64.deb
```

### Deinstallation

```sh
sudo apt purge cooler
```
